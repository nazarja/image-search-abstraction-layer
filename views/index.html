Image Search Abstraction Layer Microservice

A freeCodeCamp Javascript Back-End Challange

This challange makes use of NodeJS, ExpressJS and MongoDB.

Challange:

  • To create an Image Search Abstraction Layer Microservice. Given a search query as a parameter, a JSON object will be returned with the image URLs, alt text and page urls for a set of images relating to the given search string.
  • To be able to paginate through the responses by adding a ?offset=2 parameter to the URL.
  • Additionally, to be able to return in a seperate JSON response, a list of the most recently submitted search strings.

Usage:

• In the url bar, after the current url. Type /search/ followed by the search term you want to get images for.
• For example to search for funny cats type /search/funny cats to receive a list of images for that search term.
• If you want to paginate through extra results, you can append ?offset=2. When the offset number represents the page number you want.
• You can also see a list of the most recent search terms. Just type /recent after the current url.

Example Usage:

Click me: http://localhost:3000/search/wallpapers
Click me: http://localhost:3000/search/best memes
Click me: http://localhost:3000/search/funny gifs
Click me: http://localhost:3000/search/best imgur pics
